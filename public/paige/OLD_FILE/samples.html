Some Sample Work

Here are several different creations and works in progress.

Wordz

Wordz was a javascript collaborative project created as part of a class assignment for Professor Odeh's Game Programming for the Web. The goal was to create a fun yet relaxing abstract typing game for even the most basic of users to enjoy. We hoped that the mixture of game play and the various randomly assigned tones for each key stroke would give the player an soothing yet enjoyable gameplay.

Wordz

Subway Squeeze

Subway Squeeze was a group Game Design project as part of a Summer Game Design Workshop taken in the Summer of 2009. The goal of the project was to produce a full function game with each member undertaking individual responsibilities. I was the leading programmer and the two other members took over design and art of the project. The above link will take you to the final project submitted the last day of classes after several iterations, improvements and variations created.
